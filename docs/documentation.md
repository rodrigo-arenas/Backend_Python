General documentation and key concepts


### Status Codes

- **1xx**: Informative
- **2xx**: Success
    * 200: OK (Success)
    * 201: Created
    * 203: Non-authoritative information
    * 204: No content
- **3xx**: Redirection
- **4xx**: Client error
    * 400: bad request
    * 401: Unauthorized
    * 404: Not found
    * 409: Conflict
- **5xx**: Server error 
    * 500: Internal server error
    * 501: Not implemented
    * 502: Bad gateway
    * 503: Service unavailable
    * 504: Gateway timeout
    * 509: Network timeout
